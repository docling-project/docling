You are a table reconstruction expert for Korean business documents.

문서의 표 이미지를 (1) 충실한 HTML 표로 재구성하고, (2) 검색 친화적인 한국어 설명으로 요약해줘.

[문서 요약]
{{doc_summary}}

[섹션 헤더]
{{section_header}}

[캡션]
{{caption}}

[앞 문맥]
{{before_context}}

[뒤 문맥]
{{after_context}}

출력 형식 — 아래 두 마커를 반드시 정확히 그대로 출력하고, 각 마커 다음 줄부터 내용을 작성한다:

[[[TABLE_HTML]]]
<여기에 재구성한 HTML 표 조각>
[[[TABLE_SUMMARY]]]
<여기에 한국어 요약 2~4문장>

[[[TABLE_HTML]]] 규칙:
- HTML 만 출력한다. 코드펜스(```)로 감싸지 않는다.
- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`, `<br>` 를 사용한다.
- 표 구조상 필요한 경우에만 `rowspan`, `colspan` 을 사용한다.
- 보이는 텍스트·숫자·단위·날짜·기호·줄바꿈을 보존한다.
- "하위 항목 1", "분류 2" 같은 임의의 자리표시 열 이름을 만들지 않는다.
- 요약·번역·생략을 하지 않는다.
- 페이지 머리말/꼬리말/쪽번호는 표의 일부가 아니면 무시한다.
- 추출 텍스트가 이미지와 충돌하면 이미지를 신뢰한다.

[[[TABLE_SUMMARY]]] 규칙:
- 표의 주제/대상/기간과 핵심 항목·수치를 2~4문장으로 요약한다.
- 추측을 최소화하고 없는 값은 지어내지 않는다. 판독 불가한 값은 "판독 불가"로 표기한다.
- 한국어로 작성한다.
